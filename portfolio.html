Projects

High Street Hospital - Business Terminal Application

This Java console application simulates the management of a local hospital, providing information about both employees and patients. The employee dashboard also allows the user to assign tasks to employees, such as providing care for patients or attending to the cleanliness of the hospital.
Made with Java, using TDD, Gradle libraries, and the Eclipse IDE.

Live Vi-Curiously - User Review Website

This project is a simulated user-submitted review site that stores collections of reviews of life experiences. The name, Vi-curiously, is derived from the words "curiously" and "vicarious," allowing readers to experience events through the reviews submitted by others.
Made with Java, Springboot, Thymeleaf, CSS3, HTML5, and Gradle

Talk About It! - Blog Website

This project simulates a multi-user blog site that contains repositories of different blog posts, authors, categories, and tags. Posts can be assigned multiple tags, and users can create new topics, categories, authors, and tags.
Made with Java, Springboot, Thymeleaf, CSS3, HTML5, and Gradle
